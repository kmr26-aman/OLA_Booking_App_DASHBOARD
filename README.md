# OLA_Booking_App_DASHBOARD
This project presents an end-to-end Exploratory Data Analysis (EDA) and interactive dashboard built for an OLA-like ride booking dataset. The goal is to uncover meaningful insights into ride patterns, customer behavior, driver performance, and business metrics using data analytics and visualization tools.
